# LAGO GitHub Pages サンプル

オーナー確認用の静的サイトです。

## 公開方法

1. GitHubで新しいリポジトリを作成
2. このフォルダ内のファイルをすべてアップロード
3. GitHubの `Settings` → `Pages`
4. `Deploy from a branch` を選択
5. Branchを `main`、フォルダを `/root` にして保存
6. 数分後、GitHub PagesのURLが表示されます

## 公開前に差し替える項目

- Instagram URL
- 電話番号
- メールアドレス
- 正式な住所
- 正式な料金
- 営業時間
- GoogleマップURL
- 実際の施設写真
